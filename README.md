# 🧿 SoulRPG

A lightweight RPG system mod for Minecraft Forge 1.20.1.

## ✨ Features

- 🧠 Soul leveling system
- ⚔ Gain souls by killing mobs
- 📊 HUD overlay
- 🖥 GUI Screen
- 🌍 Full multiplayer sync
- ⚡ Clean Capability architecture

---

## 🛠 Tech Stack

- Minecraft Forge 1.20.1
- Capability API
- Custom Network Packets
- Client-Server Sync

---

## 📂 Structure

Capability-based persistent player data.

Designed for:
- Modpacks
- RPG servers
- Expansion system (skills, classes, perks)
